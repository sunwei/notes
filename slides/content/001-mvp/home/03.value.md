# MVP - 最小可行产品

目标： 让用户在喜欢的笔记软件上，编写私教服务介绍， 随时预览，一键发布。

- 用户旅程
  - 打开编辑器 - Obsidian，下载插件 - MDFriday
  - 插件安装成功后，配置部署方式和信息 (待确认)。
  - 由插件自动生成用户账号 - 用户名和账号。
  - 新建笔记
  - 在笔记上配置主题
  - 点击侧边栏预览和发布
